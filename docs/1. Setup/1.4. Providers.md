# 1.4. Providers

## Do you need provider credentials now?

No. Run `mise run check` and `mise run test` before configuring a model. These gates exercise tools, state, security callbacks, MCP, A2A construction, and adversarial cases without sending data to a provider.

You need a model only for interactive `run`/`web`, live ADK evaluation, and the full gateway lab.

## How do you configure local Qwen3?

Install Ollama from its supported distribution, then pull the Apache-2.0 Qwen3 weights:

```bash
ollama pull qwen3:4b
ollama list
```

The full local stack supplies these variables to the agent:

```bash
AGENT_GATEWAY_ENABLED=true
AGENT_MODEL=qwen3:4b
OPENAI_BASE_URL=http://agentgateway:4000/v1
OPENAI_API_KEY=agentgateway
```

For a host process talking to a locally running or port-forwarded gateway, use `http://localhost:4000/v1`. The marker API key satisfies the client contract; it is not an Ollama credential.

## How do you configure native Gemini?

Create a gitignored root `.env` from the supplied template and add only the variables for your selected Google authentication method:

```bash
cp .env.example .env
chmod 600 .env
```

For a Gemini API key:

```bash
GOOGLE_API_KEY=replace-me
AGENT_MODEL=gemini-3.5-flash
```

For Vertex AI, use the current ADK/Google Cloud documentation and Application Default Credentials on your workstation. The optional GKE overlay uses Workload Identity Federation and does not store a service-account key.

## How does configuration fail fast?

`agents/python/src/agent/config.py` parses `AGENT_*` variables into typed settings. `agents/python/src/agent/model.py` selects exactly one path:

- Gateway disabled: return the explicit native Gemini model name.
- Gateway enabled: require `OPENAI_BASE_URL` and `OPENAI_API_KEY`, then construct ADK's `OpenAILlm` adapter.

Missing gateway values raise a configuration error during model construction instead of silently falling back to another provider.

## How do you protect model data and credentials?

- Keep `.env` out of Git and restrict its permissions.
- Never bake credentials into an image, ConfigMap, command example, trace, or screenshot.
- Leave ADK/GenAI message-content capture disabled unless you have reviewed the data policy.
- Route cloud workloads through identity federation instead of long-lived service-account keys.
- Treat model prompts and tool output as potentially sensitive even in a fictional exercise.

## What is the provider checkpoint?

Choose one:

```bash
# Local path
ollama show qwen3:4b

# Native Gemini path, after configuring .env
cd agents/python && mise run run
```

For Gemini, stop the CLI after a simple read-only prompt such as `List open incidents`. Do not test a write action until Chapter 4 explains approval and audit behavior.
