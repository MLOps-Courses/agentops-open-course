# 1. Setup

Before you build an agent, you need a reproducible local environment. This chapter installs and pins every tool the course uses, sets up your language track, and configures a model provider — so from [Chapter 2](../2. Agents/) onward you can focus entirely on the agent.

Work through the shared pages, then your track (Python **or** Go), then finish the shared ones:

- **[1.0. System](./1.0. System.md)**: Prerequisites, operating-system setup, shell, and installing mise.
- **[1.1. Python](./1.1. Python.md)**: Install and manage Python with mise and uv _(Python track)_.
- **[1.2. Go](./1.2. Go.md)**: Install the Go toolchain and manage modules _(Go track)_.
- **[1.3. Containers](./1.3. Containers.md)**: Docker and Podman basics for packaging agents.
- **[1.4. Kubernetes](./1.4. Kubernetes.md)**: Stand up a local cluster with k3d, kubectl, and helm.
- **[1.5. Providers](./1.5. Providers.md)**: API keys, the `.env` file, and selecting a provider and model.
- **[1.6. Workspace](./1.6. Workspace.md)**: Git, GitHub, VS Code, and the AGENTS.md standard.
